## KEY PROTOCOLS

### AKI - KDIGO Staging

| Stage | Creatinine | Urine Output |
|-------|------------|--------------|
| 1 | 1.5-1.9x baseline OR ≥0.3 mg/dL increase | <0.5 mL/kg/h for 6-12h |
| 2 | 2.0-2.9x baseline | <0.5 mL/kg/h for ≥12h |
| 3 | 3x baseline OR ≥4.0 OR dialysis initiation | <0.3 mL/kg/h for ≥24h OR anuria ≥12h |

**AKI Workup:**
- Pre-renal: BUN/Cr >20, FeNa <1%, UOsm >500
- Intrinsic: FeNa >2%, muddy brown casts
- Post-renal: Hydronephrosis on US

**Indications for Emergent Dialysis (AEIOU):**
- **A**cidosis (pH <7.1, refractory)
- **E**lectrolytes (K >6.5, refractory)
- **I**ngestion (dialyzable toxin)
- **O**verload (pulmonary edema, refractory)
- **U**remia (encephalopathy, pericarditis, bleeding)

### CKD Staging

| Stage | GFR (mL/min/1.73m²) | Description |
|-------|---------------------|-------------|
| G1 | ≥90 | Normal/high (with kidney damage) |
| G2 | 60-89 | Mildly decreased |
| G3a | 45-59 | Mild-moderate |
| G3b | 30-44 | Moderate-severe |
| G4 | 15-29 | Severely decreased |
| G5 | <15 | Kidney failure |

**Albuminuria Categories:**
- A1: <30 mg/g (normal)
- A2: 30-300 mg/g (moderate)
- A3: >300 mg/g (severe)

### Electrolyte Disorders

**Hyponatremia:**
| Severity | Na (mEq/L) | Symptoms | Treatment |
|----------|------------|----------|-----------|
| Mild | 130-134 | Often none | Fluid restrict, treat cause |
| Moderate | 125-129 | Nausea, confusion | Fluid restrict ± salt tabs |
| Severe | <125 | Seizures, coma | Hypertonic saline |

Correction rate: ≤8-10 mEq/L per 24h (avoid ODS)
Hypertonic saline: 3% NaCl 100mL bolus over 10min, repeat x2 if severe symptoms

**Hyperkalemia:**

| K (mEq/L) | ECG Changes | Treatment |
|-----------|-------------|-----------|
| 5.5-6.0 | Peaked T waves | Kayexalate, loop diuretic |
| 6.0-6.5 | PR prolongation | + Insulin/glucose, albuterol |
| >6.5 | QRS widening | + Calcium gluconate, dialysis |

**Treatment Protocol:**
1. Calcium gluconate 1-2g IV (membrane stabilization)
2. Insulin 10U + D50 (shift K intracellular)
3. Albuterol 10-20mg nebulized
4. Kayexalate 30g PO/PR (remove K)
5. Dialysis if refractory

**Hypercalcemia:**

| Ca (mg/dL) | Symptoms | Treatment |
|------------|----------|-----------|
| 10.5-12 | Often asymptomatic | Hydration, treat cause |
| 12-14 | Fatigue, constipation | NS + furosemide |
| >14 | AMS, arrhythmia | + Bisphosphonate, calcitonin |

### Acid-Base

**Step-by-step Analysis:**
1. pH: Acidemia (<7.35) or alkalemia (>7.45)?
2. Primary disorder: Respiratory (PaCO2) or metabolic (HCO3)?
3. Compensation appropriate?
4. If metabolic acidosis: Calculate anion gap (Na - Cl - HCO3)
5. If AG elevated: Calculate delta-delta ratio

**Anion Gap Metabolic Acidosis (GOLDMARK):**
- **G**lycolene (ethylene glycol)
- **O**xoproline
- **L**actate
- **D**KA
- **M**ethanol
- **A**spirin
- **R**enal failure
- **K**etoacidosis

**Non-AG Metabolic Acidosis (HARDUPS):**
- **H**yperalimentation
- **A**cetazolamide
- **R**TA
- **D**iarrhea
- **U**reteral diversion
- **P**ancreatic fistula
- **S**aline infusion

## CRITICAL VALUES

| Finding | Action |
|---------|--------|
| K >6.5 + ECG changes | Calcium, insulin, dialysis |
| Na <120 + seizures | 3% saline bolus |
| pH <7.1 + refractory | Emergent dialysis |
| Pulmonary edema + anuria | Emergent dialysis |
| Uremic encephalopathy | Emergent dialysis |
